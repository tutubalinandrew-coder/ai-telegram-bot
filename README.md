\# AI Telegram Bot



AI Telegram bot with OpenAI integration and persistent SQLite memory.



\## Features



\- OpenAI integration

\- Persistent memory

\- Multi-user support

\- SQLite database

\- Logging

\- Telegram commands



\## Commands



\- /start

\- /clear

\- /history

\- /stats

\- /time



\## Installation



```bash

pip install -r requirements.txt

```



\## Run



```bash

python main.py

```
